接口文档

认证方式

所有 API 请求需要在 HTTP Header 中携带 API Key：

Authorization: Bearer sk-your-api-key-here

端点列表

方法路径说明
GET/v1/health健康检查
POST/v1/auth/login登录，返回 API Key
POST/v1/auth/register注册新账号
GET/v1/user/me当前用户信息
GET/v1/auth/api-keys列出 API Key
POST/v1/auth/api-keys创建新 API Key
DELETE/v1/auth/api-keys/{id}删除 API Key
GET/v1/memory/search?q=...搜索记忆
POST/v1/task?goal=...创建学习任务

帮助与反馈

快速开始

1. 注册账号并登录
2. 在 API Keys 页面创建 Key
3. 在请求头中携带 Authorization: Bearer <key>
4. 调用 /v1/ 系列 API
