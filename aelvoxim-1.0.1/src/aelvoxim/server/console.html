API 参考文档

← Home

以下为 Aelvoxim API 主要端点。完整交互式文档请访问 /docs。

方法路径说明
POST/v1/auth/loginEmail+PasswordLogin，返回 API Key
POST/v1/auth/registerSign Up新账号，返回 API Key
POST/v1/llm/chatLLM 对话（需 Bearer API Key）
GET/v1/health系统健康检查
POST/v1/task创建学习/推理任务
GET/v1/user/me当前用户信息（需 Bearer）
GET/v1/memory/{key}读取记忆（需 Bearer）
POST/v1/config配置读写（需 Bearer）

认证方式

所有 /v1/* 端点（除 auth 外）均需在 HTTP Header 中携带 Authorization: Bearer ***

在线调试器

无需离开页面即可测试 API 端点。

FAQ

API Key 无效？
从 Settings → API Keys 复制最新密钥，确保未加多余空格。
限流了怎么办？
Community 套餐每月 1,000 次调用。升级套餐可提高配额。
如何查看所有端点？
访问 /docs 查看交互式 Swagger 文档。

Integrations

← Home

Python SDK

通过 pip 安装客户端库，使用 API Key 直接调用 MetaCore 核心能力。

pip install aelvoxim-client
from aelvoxim import Aelvoxim
client = Aelvoxim(api_key="sk-...")

Webhook Config

事件驱动的回调通知，支持任务完成、知识更新等场景。

配置方式：在 Settings 中设置回调 URL，选择订阅事件类型。

REST API

完整的 RESTful API，支持任意语言调用。认证方式：Bearer Token。

curl -H "Authorization: Bearer sk-..." \
  http://localhost:9701/v1/llm/chat \
  -d '{"messages":[{"role":"user","content":"你好"}]}'
